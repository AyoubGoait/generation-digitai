# Génération digitAi

Site de la communauté Génération digitAi, consacré aux formations pratiques en intelligence artificielle.

## Déploiement GitHub Pages

Importez `index.html`, `style.css`, `script.js` et `README.md` à la racine du dépôt. Dans **Settings → Pages**, choisissez **Deploy from a branch**, puis `main` et `/root`.

Les contenus, tarifs et coordonnées sont provisoires et pourront être personnalisés progressivement.
