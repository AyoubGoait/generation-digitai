Questions fréquentes

Avant de commencer

Faut-il déjà savoir programmer ?

Non. Les notions indispensables seront expliquées progressivement. L’objectif est aussi de comprendre suffisamment le code pour mieux guider l’IA.

Quel type de projet vais-je pouvoir créer ?

Vous pourrez réaliser un site vitrine, une landing page, un portfolio ou un petit outil web adapté à votre niveau.

Ai-je besoin d’un ordinateur ?

Oui, un ordinateur et une connexion Internet stable sont fortement recommandés pour suivre les exercices dans de bonnes conditions.

L’IA fait-elle tout à ma place ?

Non. Elle accélère la production, mais vous devez définir le besoin, tester le résultat et prendre les décisions importantes.

Une attestation est-elle prévue ?

Aucune attestation n’est annoncée pour le moment.
